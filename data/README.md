# Data Inventory

## Tracked in Git

- `expression/GTEx_Analysis_2017-06-05_v8_RNASeQCv1.1.9_gene_median_tpm.gct.gz`
  Raw-ish GTEx median TPM matrix used to regenerate the reduced substantia nigra table.
- `expression/gtex_sn_expression.tsv`
  Reduced GTEx table used by the canonical analysis.
- `seeds/pd_monogenic.txt`
  Input monogenic PD seed gene symbols.
- `seeds/pd_monogenic_ensp.txt`
  Offline-mapped STRING ENSP seed identifiers.
- `seeds/pd_monogenic_seed_mapping.tsv`
  Seed-to-ENSP mapping table with the selected alias match for each seed.
- `string/9606.protein.aliases.v11.0.txt.gz`
  Compressed STRING alias table used for offline seed mapping.
- `string/9606.protein.info.v11.0.txt.gz`
  Compressed STRING protein annotation file used to map ENSP IDs to symbols.
- `string/ppi_edges.csv`
  High-confidence STRING protein edges used to rebuild the graph deterministically.

## Expected local-only or ignored files

- `string/9606.protein.links.detailed.v11.0.txt` or `.txt.gz`
  Optional raw STRING detailed interaction file. Required only for the full preprocessing rebuild.
- `string/9606.protein.aliases.v11.0.txt`
  Uncompressed alias table. The compressed `.gz` file is sufficient for the canonical pipeline.
- `string/HUMAN_9606_idmapping.dat` and `.gz`
  Present locally but not used by the canonical reproducible pipeline.
- `string/ppi_graph.pkl`
  Deterministic intermediate generated by `scripts/build_graph.py`.
- `NOT USED/`
  Archived local material not part of the canonical analysis.

## Checksum policy

`checksums.sha256` covers the reproducibility-critical tracked files. Refresh it with:

```powershell
python scripts/update_checksums.py
```
